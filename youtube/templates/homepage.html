This App Contains the following Pages

{% block sidebar %}
  • Get Api

    - This will list the stored videos in reverse chronological order of their publishing date


  • Search Api

    - This is search api which will accept either title or description as a querying parameter and returns the videos that has the query parameter either in their title or description

  • Dashboard

    A dashboard which will be having the list of all stored videos. You can filter videos either using search bar provided or you can choose the video from the available list of videos in the dropdown.
    It also has options to sort elements in ascending or descending based on the field choosen.
